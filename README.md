# VanityEngine

A browser-based, multi-threaded, high-performance vanity address generator for EVM chains (Ethereum, Base, etc.).

## Features

- **100% Client-Side:** Keys never leave your browser.
- **High Performance:** Uses WebAssembly (Rust) and Web Workers for multithreaded generation.
- **Memory Safe:** Private keys are zeroed out of memory after generation.

## Project Structure

- `/engine` - The Rust WASM engine. Handles elliptic curve cryptography and high-speed address generation.
- `/ui` - The Next.js frontend. Handles user input, multi-threading, and stats display.

## Setup Instructions

### Prerequisites
You need to have Rust, `wasm-pack`, and Node.js installed.
- Install Rust: https://rustup.rs/
- Install wasm-pack: `cargo install wasm-pack`
- Install Node.js: https://nodejs.org/

### 1. Build the WASM Engine
First, you must compile the Rust engine to WebAssembly.
```bash
cd engine
wasm-pack build --target web
```

### 2. Run the Next.js App
Once the engine is built, you can start the development server.
```bash
cd ../ui
npm install
npm run dev
```

### 3. Open the App
Open your browser and navigate to `http://localhost:3000`.

## Architecture Details

- **Concurrency:** The app uses `navigator.hardwareConcurrency` to spawn multiple Web Workers. Each worker runs its own instance of the WASM engine.
- **WASM bindings:** The Next.js app is configured with `asyncWebAssembly` to properly load the `wasm-pack` output.
- **Coordination:** The main thread coordinates the workers using a `SharedArrayBuffer` and `Atomics` to instantly signal all workers to stop as soon as a match is found.
- **Cryptography:** The engine uses the `k256` and `sha3` crates, seeded initially by the browser's `crypto.getRandomValues()`, and then utilizes `ChaCha20` inside WASM for high-speed generation.

## Testing

1. Generate an address with a short prefix (e.g. `123`).
2. Copy the generated private key.
3. Import the private key into a wallet like MetaMask.
4. Verify that the derived address in the wallet exactly matches the one generated by VanityEngine.
